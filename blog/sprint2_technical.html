Different Types of Positioning CSS

Static

Static. Generally reffered to as the "normal document flow", elements stacking and wrapping like we'd expect in a Word document.

Relative

Relative positioning allows you to move elements relative to it's already pre-existing position. It leaves a place holder to where it would normaly be so i't does'nt interupt or mess with the normal document flow.

Absolute

Absolute completely removes the element from the normal document flow. The element is positioned to and around its parent element.

Fixed

Fixed elements are positioned around the viewport, not their parent element. For example when you scroll a web page the element stays fixed in it's place on your screen/viewport.
